# راه‌اندازی سریع MedPath

پیش‌نیاز: نصب بودن **Node.js** روی سیستم.
اگر ندارید: از https://nodejs.org نسخه LTS را دانلود و نصب کنید (مرحله نصب فقط «Next» را بزنید).

---

## ویندوز (Windows)

۱. فایل zip را Extract کنید.
۲. وارد پوشه `medpath` شوید.
۳. روی فایل **`start-windows.bat`** دوبار کلیک کنید.
۴. صبر کنید تا پکیج‌ها نصب شوند (بار اول کمی طول می‌کشد).
۵. مرورگر به‌صورت خودکار باز می‌شود و آدرس `http://localhost:3000` را نشان می‌دهد.
۶. برای توقف، پنجره سیاه (ترمینال) را ببندید.

> اگر ویندوز پیام «Windows protected your PC» داد، روی **More info** سپس **Run anyway** کلیک کنید.

---

## مک / لینوکس (macOS / Linux)

۱. فایل zip را Extract کنید.
۲. ترمینال را باز کنید و وارد پوشه `medpath` شوید:
   ```bash
   cd medpath
   ```
۳. این دستور را یک‌بار اجرا کنید تا فایل قابل اجرا شود:
   ```bash
   chmod +x start-mac-linux.sh
   ```
۴. اسکریپت را اجرا کنید:
   ```bash
   ./start-mac-linux.sh
   ```
۵. صبر کنید تا پکیج‌ها نصب شوند، سپس مرورگر به‌صورت خودکار باز می‌شود.
۶. برای توقف، `Ctrl + C` را بزنید.

---

## نکات

- بار اول، نصب پکیج‌ها ممکن است **۲ تا ۵ دقیقه** طول بکشد (بسته به سرعت اینترنت).
- در دفعات بعد که اسکریپت را اجرا کنید، چون پکیج‌ها قبلاً نصب شده‌اند، سریع‌تر اجرا می‌شود.
- اگر مرورگر خودکار باز نشد، خودتان به آدرس زیر بروید:
  ```
  http://localhost:3000
  ```
